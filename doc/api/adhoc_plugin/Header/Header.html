Creates a Header object.

The messageType indicates the type of message exchanged. The label uniquely identifies the device along with its name and MAC address mac.

The address represents the device logical address (UUID for Bluetooth LE and IP for Wi-Fi).

The type of technology used by the device is determined by deviceType. It can be either Bluetooth Low Energy or Wi-Fi Direct.
